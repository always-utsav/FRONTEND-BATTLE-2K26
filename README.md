# MITS Cinematic Portal

## 📖 Overview
A hyper-immersive, Awwwards-style student portal built exclusively for Madhav Institute of Technology & Science (MITS). Breaking away from traditional, static academic websites, this project utilizes a dynamic "scrollytelling" architecture to deliver a premium, Apple-inspired cinematic experience while retaining total academic utility for students.

## 🚀 Unique Selling Proposition (USP)
**"Transforming Academic Utility into an Awwwards-Tier Cinematic Experience."**
This application bridges the gap between high-end digital agency portfolios and strict institutional necessity. It combines a lag-free, 60fps canvas-driven scroll sequence with instant, latency-free access to critical academic resources.

## ✨ Special Features
- **Scroll-Linked Cinematic Hero:** Replaces clunky MP4 videos with a heavy-duty 288-frame `<canvas>` image sequence that scrubs forwards and backwards mapped perfectly to the user's scroll depth.
- **Autonomous Ambient Audio Engine:** An intelligent cinematic drone track that aggressively bypasses browser autoplay blocks via broad-spectrum interaction listeners, and subsequently fades out exactly at 80% scroll depth to prevent overlapping noise while reading.
- **The "Chameleon" Navbar:** A state-aware navigation header that fluidly cross-fades between a completely transparent overlay (with white text) over dark video backgrounds, into a frosted-glass blurred navbar (with navy text) over white backgrounds.
- **Hand-Drawn Micro-Interactions:** Custom SVG path animations (spirals, underlines) that elegantly draw themselves upon scroll-reveal to highlight key content without relying on heavy UI libraries.
- **Zero-Latency Academics Grid:** A highly responsive React state filtering infrastructure providing students instant, tab-less access to Forms, Syllabi, Resource Hubs, and Layouts.

## 💻 Tech Stack Highlights
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS (Glassmorphism & Depth Tokens)
- **Motion:** Framer Motion (Spring-physics UI transitions)
- **Rendering:** Vanilla HTML5 `<canvas>` + `requestAnimationFrame`
